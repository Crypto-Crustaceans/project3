About

Our goal was to create a model that could make accurate predictions for the price of Bitcoin.
We trained our model using historical Bitcoin market data, blockchain data, and social data,
as well as using the results of a sentiment analysis of Bitcoin news headlines.

CryptoCompare API

We used the cryptocompare.com API to retrieve historical market, blockchain, and social data, and we scraped news.bitcoin.com to obtain historical news headlines. We also use the API to retrieve current data and recent news articles, so that the model can make live predictions.

Scikit-learn & NLTK

Scikit-learn is a ML library featuring various classification, regression, and clustering algorithms. We used historical data to train a regression algorithm (to make price predictions) and a classification algorithm (to determine the postive, negative, or neutral sentiment of news headlines). NLTK is a suite of NLP libraries, which we used as part of our preprocessing pipeline.

JavaScript

D3.js combines powerful visualization components and a data-driven approach to DOM manipulation to create custom, dynamic, and interactive data visualizations. We used d3.js to create a calendar visualization showing how our model's predictions using test data compared to actual historical prices.

Prediction Model

This calendar visualizes the results of our prediction model using test data.

Each square shows the historical price of Bitcoin with a 1-day or 7-day offset for any given date (bottom-left corner) as well as the output of our prediction model using test data (top-right corner). Green blocks represent positive price movement and pink blocks represent negative price movement.

The visualization was created using d3.js and can be filtered to show the fit of our 1-day or 7-day prediction model.

Project Timeline

  • Getting Data:

    CryptoCompare API
    & Web Scraping

    We used the cryptocompare.com API to retrieve historical Bitcoin market data (daily open and close prices, volume etc.), blockchain data (number of transactions, number of large transactions, transaction value, etc.), social data (e.g., number of reddit posts, etc.). The historical data retrieved via the API went back several years.

    We created a web scraping tool to grab thousands of historical Bitcoin news headlines going back several years from news.bitcoin.com.

    We also used the cryptocompare.com API to retrieve current market data (including real-time price), blockchain data, social data, and news articles.

  • Data Clean Up & Preprocessing:

    We cleaned, merged, and aggregated the historical market, blockchain, and social data to create our training data.

    Additionally, we used NLTK to perform preprocessing steps on our historical news headlines including tokenizing, stemming, removing punctuation and numbers, formatting the text as lowercase, excluding stop-words, and finally vectorizing the text.

    These preprocessing steps were necessary to feed the headlines to the classification algorithm. The results of the classification algorithm’s sentiment analysis were also merged into our training data.

  • Creating the Model:

    SVM Classifier &
    Random Forest Regressor

    After preprocessing, the news headlines were split into training and test sets to evaluate the performance of various classifiers. The SVM classifier performed best, and we chose it as the final classification algorithm.

    The final merged historical dataset was then split into training and test sets to evaluate the performance of various regressors. The Random Forest regressor performed best, and we chose it as the final regression algorithm.

    Additionally, we performed grid searches to hyper-tune the models and select the best algorithm parameters. Once finalized, the trained algorithms were saved for future use.

  • The Final App:

    Flask Server

    We created a Flask application to host a web page and make live Bitcoin price predictions.

    The application retrieves current data from the cryptocompare.com API, and the data is fed to our pre-trained algorithms to make live predictions. D3.js is used to update this web page with the real-time Bitcoin price as well as our 1-day and 7-day price predictions. D3.js is also used to create the calendar visualization on this web page.
